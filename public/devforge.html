AAD Users

Admin dashboard users are added through Microsoft Entra ID, not local passwords.

How to add an admin:

  1. Add or invite the user in Microsoft Entra ID / the Azure AD app assignment.
  2. Add the email or object ID to AZURE_AD_ADMIN_EMAILS or AZURE_AD_ADMIN_OBJECT_IDS.
  3. The user signs in with AAD. Their admin record is provisioned automatically.
